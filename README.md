# Academic ERP Portal

A full-stack Academic ERP Portal built using React.js, Node.js, Express.js, and MongoDB Atlas.

## Features

- JWT Authentication
- Student, Faculty, and Admin Dashboards
- Course Enrollment System
- Course Drop Functionality
- Semester-wise Results Module
- Role-Based Access Control
- Responsive User Interface

## Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- MongoDB Atlas

## Installation

### Client

```bash
cd client
npm install
npm start
```

### Server

```bash
cd server
npm install
npm start
```
